  • 01企业AI协作全景认知：AI如何改变团队工作模式25分钟▼
    情境导入
    传统企业协作痛点：审批流程长、信息孤岛、重复沟通成本高、跨部门协同困难。展示一个典型场景：市场部要用AI做活动素材，但IT部的AI权限未打通。
    概念拆解
    AI协作三要素：工具层（各类AI工具的能力边界）、流程层（AI介入现有工作流的节点选择）、人机层（人决策vsAI辅助的分工原则）。企业AI成熟度四阶段：单点试用→部门推广→跨部门协作→全企业AI生态。
    分步实操
    步骤1：绘制本企业现有协作流程图 / 步骤2：标注各节点AI可介入环节 / 步骤3：建立跨部门AI共享知识库 / 步骤4：制定AI协作基础规范文档
    跟练
    选一个企业内跨部门流程（如报销审批），用AI生成协作流程图，标注出AI可替代的节点，计算可节省工时。

  • 05AI驱动企业审批流程自动化设计26分钟▼
    情境导入
    传统企业一张报销单要经过5道审批，平均耗时3天。采购审批更复杂，涉及预算、合规、供应商三方审核。AI可以自动化大部分审核节点。
    概念拆解
    审批流程AI化三原则：规则化→所有可量化的审核标准固化进AI；分级化→金额/风险度自动分级路由到不同审批人；透明化→AI判断逻辑可见可追溯。AI审批边界：AI只做规则校验，最终决策权保留在审批人。
    分步实操
    步骤1：梳理企业全部审批流程，标注每个节点的审核规则 / 步骤2：区分AI可自动审批vs需人工审批的节点 / 步骤3：设置审批规则和阈值 / 步骤4：搭建AI审批看板
    跟练
    为一家企业设计差旅报销AI审批流程：写出每个节点的审核规则、AI自动审批的条件、异常情况人工介入机制。
    课后接单实操工单
    企业审批流程优化方案：梳理一家50人公司的采购+报销+请假三条审批流程，设计AI自动审批规则和阈值，输出完整流程图和配置方案。

  • 06多部门AI协同项目管理实战30分钟▼
    情境导入
    市场部要上线一个营销活动，涉及市场（方案）、设计（物料）、技术（落地页）、运营（推广）4个部门。传统方式：微信群来回沟通、文件版本混乱、进度完全靠催。
    概念拆解
    AI协同项目管理的核心：统一信息中枢——AI驱动的项目看板自动同步各部门进度；自动阻断依赖——当一个部门的产出依赖于另一个部门时，AI自动追踪前置任务状态；风险预警——AI根据历史数据预测延期风险。
    分步实操
    步骤1：定义项目模版（部门角色、任务依赖关系、交付物标准）/ 步骤2：AI自动创建项目空间并分配任务 / 步骤3：配置自动化规则（任务完成自动通知下游）/ 步骤4：AI生成项目日报周报
    跟练
    为一个新品上市项目（涉及4个部门）设计AI协作方案：绘制部门依赖图、配置自动化规则、设定AI预警条件。
    课后接单实操工单

    分步实操
    步骤1：搭建企业统一文档协作平台 / 步骤2：设置文档分类体系与命名规范 / 步骤3：配置AI自动版本号+变更记录 / 步骤4：AI自动检测跨部门文档冲突并提醒
    跟练
    为一家人力资源+法务+财务经常交叉协作的企业设计文档协同方案，包含版本规则、权限矩阵、AI冲突检测逻辑。

  • 08企业AI协作SOP标准化体系建设22分钟▼
    情境导入
    企业上了AI工具后一片混乱：有人用AI做了80%的工作但没人审核直接交付给客户出了问题；有人完全不用AI导致效率被甩开。缺少标准化的AI协作流程。
    概念拆解
    AI协作SOP核心模块：AI使用范围清单（哪些工作必须/可以/禁止用AI）、AI产出审核标准（准确性/合规性/品牌一致性检查项）、人机交接标准（AI产出到什么程度可以交接给人审核）、异常处理流程（AI出错时的责任追溯和修正机制）。
    分步实操
    步骤1：梳理企业全部工作场景，分类标注AI可用性 / 步骤2：定义AI产出物质量检查清单 / 步骤3：建立人机协作SOP文档模板 / 步骤4：定期复盘AI协作效果并更新SOP
    跟练
    为一家电商企业设计AI协作SOP：包含客服回复、商品文案、数据分析三个场景的AI使用规范和审核标准。
    课后接单实操工单
